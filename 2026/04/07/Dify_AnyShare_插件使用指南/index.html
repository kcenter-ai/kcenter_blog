概述

AnyShare Dify 插件是一款用于从 AnyShare 文档库中召回文档切片的工具插件。通过该插件，您可以在 Dify 工作流中集成 AnyShare 的文档检索能力，实现基于企业文档知识的 RAG（检索增强生成）应用。

主要功能

  • 从 AnyShare 文档库中检索相关文档切片
  • 支持切片检索
  • 可与企业现有文档权限体系集成
  • 支持在工作流中灵活配置召回参数

前置准备

在使用 AnyShare Dify 插件之前，您需要提前准备好以下信息：

1. 应用账户凭证

参数 说明 获取方式

应用账户密钥 用于身份认证 联系 AnyShare 管理员申请

2. 文档库权限

确保为应用账户分配了对应的文档库权限，且该文档库满足以下条件：

  • 已开启全文解析功能
  • 已开启向量切片功能

插件下载与安装

插件包

插件包文件：anyshare_retrieval.difypkg

插件包下载：anyshare_retrieval.difypkg

安装步骤（本地文件方式）

  1. 登录 Dify 平台

  2. 进入 插件管理 页面

    • 点击 Dify 平台右上角的「插件」按钮
    • 进入插件管理页面

  3. 选择 上传本地文件 方式

  4. 上传 anyshare_retrieval.difypkg 文件

  5. 等待插件安装完成

关闭第三方插件验证

⚠️ 重要：在某些 Dify 配置下，可能需要关闭第三方插件验证才能正常使用。

操作步骤

  1. 进入 Dify 系统设置

  2. 找到 插件安全设置 或 第三方插件验证 相关选项

  3. 关闭第三方插件验证功能

  4. 保存设置

Dify 本地插件安装指导

离线/本地安装

  1. 从本地上传 .difypkg 文件
  2. 点击「上传本地文件」按钮
  3. 选择已下载的插件包
  4. 确认安装

常见问题

Q1: 插件安装失败怎么办？

解决方案：

  1. 确认 .difypkg 文件完整未损坏
  2. 关闭第三方插件验证
  3. 检查 Dify 版本是否支持该插件

Q2: 召回结果为空？

可能原因：

  1. 应用账户没有文档库权限
  2. 文档库未开启全文解析或向量切片

解决方案：

  1. 联系管理员确认权限配置
  2. 确认文档库已正确配置

Q3: 召回结果不准确？

解决方案：

  1. 调整 limit 参数，增加召回数量
  2. 优化查询问题的表述

Q4: 如何确认文档库已开启全文解析和向量切片？

检查方式：

  1. 登录 AnyShare 管理控制台
  2. 进入智能搜索-文档解析策略
  3. 确认对应文档库或文件目录的「全文解析」和「向量切片」功能已启用

Q5: 应用账户 ID 和密钥如何获取？

获取方式：
联系 AnyShare 管理员申请，需要提供：

  • 应用名称
  • 需要的文档库权限范围
  • 预期用途说明
